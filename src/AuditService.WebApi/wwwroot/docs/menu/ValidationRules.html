Validation rules
In .NET, type casting is automatic. Therefore, if you specify a value in the filter field that does not correspond to the type, then it will be ignored. Automatic type casting is described below. Be careful!
Format The field can be null Required field
GUID NULL 00000000-0000-0000-0000-000000000000
DateTime NULL 01/01/0001
Int NULL 0
When specifying the date in the wrong format (filter.Date= 11ay.sut1.y12/14) the server will treat this as null if the field is nullable. See source for details: docs.microsoft.com
