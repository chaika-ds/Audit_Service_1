-- Настройка OffsetExplorer для локального приема сообщений --
------------------------------------------------------------------------------------------------------------

1. Вкладка Security: Type - Plaintext
2. Вкладка Advanced: Bootstrap servers - localhost:9092

Для работы с приемом сообщений в Кафку нужны установленные локально сервера: 
	Kafka: порт 9092
	Zookeeper: порт 2181  

------------------------------------------------------------------------------------------------------------
- Запись сообщений в Kafka
------------------------------------------------------------------------------------------------------------

1. Для записи сообщений в кафку - Producer в виде интеграционного теста в классе ProducerAuditLogGenerator
2. Создание сообщений аудит лога в кафку - в классе AuditLogMessageDtoBuilder, настройка отправляемых данных - в классе BuilderBase
